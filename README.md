# 45 Days Training - Python & AI

This repository contains code, notebooks, and resources created during a **45-day training program on Python and Artificial Intelligence**.  
It includes practice exercises, examples, and mini-projects covering Python fundamentals, data handling, and AI concepts.

---

## 📂 Project Structure
- **abstract.ipynb** → Demonstrates Python OOP concepts using `ABC` and `abstractmethod`.
- **filehandle.ipynb** → File handling operations in Python.
- **numpy.ipynb** → Practice with NumPy arrays and numerical computations.
- **data.txt** → Sample dataset for file handling and data processing.
- **function/** → Folder containing additional Python functions and exercises.
